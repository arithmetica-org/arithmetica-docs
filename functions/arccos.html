Arithmetica: Documentation

Function Stubs:

C
char *arccos (const char *number, size_t accuracy);

C++
std::string arccos (std::string n, size_t accuracy);

Description:
Computes the inverse cosine of a number less than 1.

Parameters:
const char * number: The number to compute the inverse cosine of.
size_t accuracy: The number of decimal places to calculate accurately to.

Return Value: The inverse cosine of the first argument.

Example Usage:

C 
#include <arithmetica.h>
#include <stdio.h>
#include <stdlib.h>

int main() {
  char *x = arccos("0.5", 10);
  printf("arccos(0.5) = %s\n", x);

  free(x);
  return 0;
}
Output:
arccos(0.5) = 1.0471975512

C++ 
#include <arithmetica.hpp>
#include <iostream>

int main() {
  std::cout << "arccos(0.5) = " << arithmetica::arccos("0.5", 10) << "\n";

  return 0;
}
Output:
arccos(0.5) = 1.0471975512
