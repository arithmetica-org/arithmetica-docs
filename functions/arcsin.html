C
char *arcsin (const char *number, size_t accuracy)

C++
std::string arcsin (std::string n, size_t accuracy);

Description:
Computes the inverse sine of a number less than 1.

Parameters:
const char *number: The number to compute the inverse sine of.
size_t accuracy: The number of decimal places to calculate accurately.

Return Value:
The inverse sine of the first argument.

Example Usage:

C 
#include <arithmetica.h>
#include <stdio.h>
#include <stdlib.h>

int main() {
  char *x = arcsin("0.5", 10);
  printf("arcsin(0.5) = %s\n", x);

  free(x);
  return 0;
}
C++ 
#include <arithmetica.hpp>
#include <iostream>

int main() {
  std::cout << "arcsin(0.5) = " << arithmetica::arcsin("0.5", 10) << "\n";

  return 0;
}

import arithmetica

How it works
The code uses this infinite series to calculate the inverse sine of a number: \[\sin^{-1}{x} = \sum_{n=0}^{\infty} \frac{(2n)!}{(2^n \cdot n!)^2} \cdot \frac{x^{2n+1}}{2n+1}\] From this, a formula that relates the nth term to the (n+1)th term can be derived: \[t_{n+1} = t_n \frac{(x(2n-1))^2}{2n (2n+1)} \] This formula is used to calculate the next term in the series, and the series is continued until the desired accuracy is reached.
